Volume of Ingress Octets per Second

You must use a browser that supports the canvas tag to view this example.

This is a demonstration of how a Boundary dataSource interacts with a custom visualization. In this example, we are displaying the amount of ingress octets measured per second for the last 600 seconds across all meters in an organization.

Activating the "unsubscribe" button will cause the graph object to stop recieving updates from it's dataSource. Note that when a dataSource has no subscribers, it will become idle. Activating the "subscribe" button will cause the graph object to begin recieving updates again.

See the example-volume-graph.js file for implementation details.
